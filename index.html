Planejamento do Projeto REA

Algoritmos em Ação

1. Tema do Projeto

Disciplina Base: Lógica de Programação

Descrição: Este projeto tem como objetivo criar um Recurso Educacional Aberto (REA) interativo que auxilie estudantes iniciantes a compreenderem os fundamentos da lógica de programação por meio de uma abordagem prática, visual e lúdica.

Tópicos Abordados:

  • Declaração e uso de variáveis e constantes
  • Operadores aritméticos, relacionais e lógicos
  • Comandos de entrada e saída
  • Estruturas de decisão (if/else)
  • Estruturas de repetição (for, while)
  • Vetores e funções básicas

2. Formato do Conteúdo

  • Vídeos Educacionais: Conteúdo explicativo sobre os tópicos abordados, com exemplos práticos e animações.
  • Aplicação Web Interativa (SPA): Exercícios programáveis e simulações de algoritmos.
  • Repositório GitHub: Código-fonte aberto e documentado.
  • Publicação: Site hospedado via GitHub Pages para livre acesso.

3. Tecnologias Utilizadas

  • Frontend: HTML5, CSS3, JavaScript com React.js
  • Componentes UI: Material UI
  • Hospedagem: GitHub Pages
  • Prototipação (opcional): Figma para definição de layout

4. Objetivos Educacionais

  • Proporcionar um material didático acessível, aberto e reutilizável sobre lógica de programação
  • Facilitar o entendimento de conceitos fundamentais através da prática e visualização
  • Desenvolver a autonomia do estudante por meio de feedbacks imediatos
  • Divulgar o conhecimento gerado por alunos e professores da instituição

5. Desenvolvimento de Competências

  • Interpretação de Problemas: Identificação das dificuldades mais comuns no aprendizado de lógica
  • Modelagem: Planejamento dos algoritmos e simulações aplicadas nos exercícios
  • Resolução Computacional: Implementação dos conceitos com lógica estruturada e interativa
  • Validação: Testes, refinamento e coleta de feedback dos usuários
